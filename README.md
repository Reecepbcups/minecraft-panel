# Setup
### Arch
```
pacman -S ufw python-pip sudo screen
python -m pip install -r requirements.txt
```

### Ubuntu / Debian
```
sudo apt install ufw python3-pip sudo screen
python3 -m pip install -r requirements.txt
```
